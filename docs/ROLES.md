# Roles

Roles are defined as follows:

---

## 1. Admin

- All access as **Contract Manager**
- Timesheet access with rates
- Invoices file/folder
- Can **publish / unpublish** work orders

---

## 2. Contract Manager

- Work order prices
- Can create work orders (WRS or General)
- Can **publish / unpublish** work orders
- Who accepted
- Only check-in / check-out (no photos, no location)
- All projects
- All work orders
- All prices
- Can modify any ongoing work order

---

## 3. Supervisor

- Can create work orders but without prices (notification) — only tools, will not see prices
- Can **publish / unpublish** work orders that they created
- Can only adjust work orders that they created (there can be multiple work orders for the same flat — possibly a separate window)
- Can see all projects
- All work orders from the assigned project

---

## 4. WRS Subcontractors

> Admin can select the users that will have access to the following:

- Can see all projects
- After choosing a project **and** checking in, can only see flats/work orders from that project
- Cannot create/modify work orders and cannot see prices
- Does not send work updates at all
- Only sends check-in or check-out notifications, but only to Marius

---

## 5. General Subcontractor

- Can see all projects
- Choose project + check-in
- Upload photos
- Check-out
